# Simple-CPUSim
a lil simple simulation show how a CPU work

This project is open-source and also it's a place so you can play with it or make your own opcode as well if you interested it. This project is my custom CPU with my own custom ASM (Assembly) idea.
And the opcode is simple so you can code yourself something you like in the text file (.txt) or even upgrade it if it isn't quite right, i also appreciate if you want to upgrade it a bit tho.

I also loved to explore somethings that make us today, so this project may have a bit headache (i also), but it was fun when you actually done one things like coding in ASM.

Instruction format:<br>
opcode r0 r1 r2
<br>or<br>
opcode operand0 operand1 operand2

Features:<br>
INPUT: handle input from keyboard and assign the character by convert into ASCII number.<br>
SCR: Assign color value so you can actually "draw" a "pixel" on console.<br>
DSCR: Display/output the assigned value on console.<br>

I hope you enjoy this project alone.
